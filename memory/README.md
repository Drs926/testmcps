Memory V1

Purpose
- Provide a small, stable knowledge base for this repo.
- Keep content human-editable and easy to scan.

Rules
- Read-only after initial generation (except manual edits by the user).
- The MCP server never writes to /memory.
- Cache and DB only live in .mcp-cache/ (gitignored).
- Reindex is manual only.

Files
- MEMORY.md: concise summary and constraints.
- DECISIONS.log.md: decision log (dated entries).
- INCIDENTS.log.md: incident log (dated entries).
- CONTEXT_PACK.md: quick-start context.
- INDEX/repo_map.json: repo map metadata.
- INDEX/summaries.md: short summaries.
